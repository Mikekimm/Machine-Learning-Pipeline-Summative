# Screenshot Evidence Checklist

Place your screenshots in this folder using these exact names:

1. health-endpoint.png
2. metrics-endpoint.png
3. deployment-proof-readme.png
4. notebook-outputs.png

Recommended capture targets:
- Health endpoint: https://machine-learning-pipeline-summative.onrender.com/health
- Metrics endpoint: https://machine-learning-pipeline-summative.onrender.com/metrics
- Deployment proof section in the main README
- Notebook outputs showing training and metrics

After placing screenshots, commit and push:

git add evidence/screenshots
git commit -m "Add submission evidence screenshots"
git push
